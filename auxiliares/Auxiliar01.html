Curvas

Polyscope nos permite tambin dibujar curvas, que viene a ser parecido a dibujar lneas en OpenGL. Necesitamos los puntos en el espacio y cmo se conectan.

Como ejercicio de este auxiliar, dibujaremos un crculo en nuestro espacio 3D. Pero necesitaremos repasar un poco de geometra y lgebra lineal.

Obtener puntos de un crculo

Los crculos son objetos que viven en un plano, pero en un espacio 3D este puede estar inclinado, as que nuestro crculo estar definido por 3 valores:

  • Centro
  • Radio
  • Normal al plano del crculo

El centro y radio del crculo nos permitirn entenderlo dentro del plano, y es la normal la que nos permite fcilmente describir el plano perpendicular, pues con ella podemos obtener los dos vectores del plano.

Claro que podramos directamente definir el crculo entregndole estos dos vectores, pero es ms complicado de utilizar a la hora de querer uno manualmente definir crculos.

Obtencin de los vectores perpendiculares a la normal

Para obtener el primer vector del plano, llammosle $v_1$, necesitamos que sea perpendicular a la normal. Pero recordemos que el producto punto entre dos vectores perpendiculares es siempre 0, por lo que podramos generar uno en base a esta regla.

Si $n=(a, b, c)$, entonces basta que $v1=(-b, a, 0)$, pues esto har que $n\cdot v = 0$. Aunque hay una excepcin, si la normal fuera $(0, 0, 1)$ esto no funcionara, sin embargo en este caso podemos libremente elegir el vector $(1, 0, 0)$, que sabemos que es perpendicular.

Ahora que tenemos $v1$, como sabemos que el producto cruz entre dos vectores perpendiculares produce uno perpendicular a ambos, basta $v_2 = v_1\times n$

Obtencin de los puntos

Normalmente en 2D, sabemos que podemos obtener puntos de un crculo en base a un ngulo, utilizando $(R\cos\theta, R\sin\theta) = R\cos\theta\hat{i}+ R\sin\theta\hat{j}$. En este caso, como nuestras bases son $v1$ y $v2$, utilizamos simplemente $$ \vec{C} + R\cos\theta\vec{v1}+ R\sin\theta\vec{v2} $$

Notemos que C es para mover el crculo a su centro, el punto C.

# Curve network examplee

import numpy as np
import polyscope as ps

class Circle:
    def __init__(self, center, radius, normal):
        self.c = np.array(center)
        self.r = radius
        self.n = np.array(normal)
        self.n = self.n/np.linalg.norm(self.n)  # normalized here so we don't have to worry
        

def generate_circle_node_edges(circle: Circle, n_nodes=10):
    c, r, n = circle.c, circle.r, circle.n
    
    # let v1, v2, n an orthonormal system
    
    # get v1 so <v1, n> == 0
    v1 = np.array([n[1], -n[0], 0])
    if np.array_equal(v1, np.zeros(3)):
        v1 = np.array([1, 0, 0])
    
    # v2 = n x v1
    v2 = np.cross(n, v1)
    
    # make them orthonormal
    v1 = v1/np.linalg.norm(v1)
    v2 = v2/np.linalg.norm(v2)
    
    # sample
    nodes = []
    edges = []
    for i in range(0, n_nodes):
        theta = 2 * np.pi * (i / n_nodes)
        nodes.append(c + r * (v1 * np.cos(theta) + v2 * np.sin(theta)))
        edges.append([i, (i+1) % n_nodes])
    
    return np.array(nodes), np.array(edges)

our_circle = Circle([0, 0, 0], 10, [0, 1, 1])
circle_nodes, circle_edges = generate_circle_node_edges(our_circle, 20)


ps.init()
ps_cloud = ps.register_point_cloud = ps.register_point_cloud("Supporting Circle Centers", np.array([our_circle.c]))
ps.register_curve_network("Circle", circle_nodes, circle_edges, radius=0.001)
ps_cloud.add_vector_quantity("Normal", np.array([our_circle.n]))

ps.show()

Vemos que adicionalmente a registrar la curva con los nodos y las aristas, registramos el centro del crculo y le asociamos su normal en una cantidad vectorial, para poder visualizar completamente nuestro crculo.
